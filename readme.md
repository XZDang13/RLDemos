1.Create a Python Virtual Environment
    To avoid dependency conflicts between Python packages, it’s recommended to use a virtual environment. The method to create one may vary depending on your Python setup.

2. Set Up the Virtual Environment
    Install Pytorch, gymnasium.
    Install RLAlg: pip install git+https://github.com/XZDang13/RLAlg.git

3. Run Demos and Tune Hyperparameters
    Run the provided demo scripts in RLAlg to understand how the algorithms work. Try identifying key hyperparameters (e.g., learning rate, discount factor, batch size) and fine-tune them to improve performance.

4. Try PPO on LunarLander
    Use the Proximal Policy Optimization (PPO) algorithm on the LunarLander environment from Gymnasium.
